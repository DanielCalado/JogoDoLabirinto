# JogoDoLabirinto
O jogo é um jogo de Labirinto na qual foi feito na linguagem C++, foi utilizado uma placa mbed e dois joysticks para controlar os movimentos dos jogadores. Possui dois jogadores que iniciam em posições aleatórias e seu objetivo é que quem chegar primeiro no centro, vence o jogo.
